# Code of Conduct

## Our Pledge

We pledge to make participation in this project respectful, constructive, and harassment-free for everyone.

## Expected Behavior

Participants are expected to:

- Be respectful and professional.
- Give constructive feedback.
- Focus on evidence, safety, and technical quality.
- Respect differing viewpoints and experience levels.
- Avoid personal attacks, harassment, or discriminatory language.

## Unacceptable Behavior

Unacceptable behavior includes:

- Harassment, intimidation, or abuse.
- Discriminatory comments or conduct.
- Publishing private information without permission.
- Trolling, insults, or sustained disruptive behavior.
- Deliberately weakening safety, verification, or security expectations without clear review.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, pull requests, or other contributions that are not aligned with this Code of Conduct.

## Scope

This Code of Conduct applies to project spaces, including issues, pull requests, discussions, documentation, and related community channels.
